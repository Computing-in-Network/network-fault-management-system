# v0.2.2 发布说明

发布日期：2026-02-23

## 本版本重点

- 完成拓扑接入策略修正：
  - 卫星可同时连接多个移动节点（飞机/船只）
  - 每个移动节点在同一时刻仅允许连接 1 颗卫星
- 补充单元测试，确保该约束可回归验证。
- 更新确定性快照基线，保证回归测试稳定。

## 兼容性

- 不新增接口，不改变 WebSocket/Redis 数据结构。
- 该版本属于链路选择语义修正，可能导致以下指标较旧版本下降：
  - `edge_count`
  - `avg_degree`
  - `mobile_connected_ratio`
- 上述变化为预期行为，不属于性能退化。

## 验证

- 后端：`PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 uv run python -m pytest -q`
- 前端：`cd frontend && npm run build`
- 规则校验：移动节点卫星链路度数最大值为 `1`，卫星仍可与多个移动节点建链。
