# v0.2.1 发布说明

发布日期：2026-02-23

## 本版本重点

- 前端构建分包优化：
  - `index`（业务入口）
  - `vendor`（通用三方依赖）
  - `vendor-cesium`（Cesium 相关依赖）
- 主入口包体显著下降，利于加载与缓存策略。

## 兼容性

- 不变更后端接口与业务语义。
- 仅调整构建产物分包策略。

## 验证

- 后端：`PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 uv run python -m pytest -q`
- 前端：`cd frontend && npm run build`
