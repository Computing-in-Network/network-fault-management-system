# v0.2.0 发布说明

发布日期：2026-02-23

## 本版本重点

1. 拓扑能力增强
- 链路策略参数外置 + 热加载
- 链路稳定性增强（最短保持时长、翻转指标）
- 连通性统计（分量、最大分量占比、直径近似）
- 增量几何重算框架（默认关闭）

2. 前端可用性提升
- 图层控制面板与偏好持久化
- 节点/链路拾取高亮 + 详情侧栏
- 运行告警（断流、延迟、低帧率）
- 时间控制（暂停/单步/倍速）

3. 质量与流程
- 固定种子快照基线回归
- Git Flow 回退标签规范与脚本

## 关键兼容性

- 增量几何模式默认关闭，不影响既有行为。
- 指标字段新增为向后兼容扩展，不删除旧字段。

## 发布前验证

- 后端：`PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 uv run python -m pytest -q`
- 前端：`cd frontend && npm run build`

## 回滚

- 开发分支快照：`rb-develop-*`
- 正式版本：`vX.Y.Z`
- 详见：`docs/gitflow_rollback.md`
