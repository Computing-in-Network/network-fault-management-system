# v0.3.0 发布说明

发布日期：2026-02-23

## 本版本重点

- 新增网络健康不平衡度指标 `I(QoE-Imbalance)`：
  - 后端按最短跳数 + Sigmoid 映射 + 熵归一化实时计算；
  - 前端 HUD 实时展示指标值，便于联调与论文指标对照。
- 修复故障面板可用性问题：
  - 故障条目过多时可滚动浏览；
  - “解除全部故障”操作保持在底部固定区域，始终可点击。

## 兼容性

- 不新增外部接口，不改变现有 WebSocket 消息结构字段。
- 仅在 `metrics` 中新增 `qoe_imbalance` 指标字段，旧前端不会因该字段新增而中断。

## 验证

- 后端：`PYTEST_DISABLE_PLUGIN_AUTOLOAD=1 uv run pytest tests/test_topology.py tests/test_stream_server.py -q`
- 前端：`cd frontend && npm run build`
- 人工验收：注入大量故障条目时，故障面板可滚动到末尾并执行“解除全部故障”。
